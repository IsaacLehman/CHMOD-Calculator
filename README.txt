Description:
  Allows you to quickly calculate Linux filesystem permissions as used by the
  chmod command line utility.

How to use:
  Check the desired boxes or directly enter a valid numeric value (e.g. 777) or
  symbolic notation (e.g. rwxrwxrwx) to see its value in other formats.

File Permissions:
  File permissions in Linux file system are managed in three distinct user
  classes: user/owner, group and others/public. Each class can have read, write
  and execute permissions. File permission can be represented in a symbolic or
  numeric (octal) format.
